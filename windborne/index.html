Project: Life Garden

An interactive ecosystem simulator

Try it | Source

For a non-technical audience

I built an interactive ecosystem simulator. You place species — grass, flowers, bees, rabbits — and watch them interact according to simple rules that produce complex emergent behavior. Bees pollinate flowers, rabbits eat grass, populations boom and crash. It's a toy model of how local interactions create large-scale patterns, which is fundamentally what weather systems do too.

For a technical peer

A multi-species cellular automaton — vanilla JS, canvas-rendered, no framework. Each species has distinct survival, reproduction, and aging rules, plus cross-species interactions (pollination, grazing). Supports step-back with full history. I vibe-coded it for fun because I like systems where local rules produce global consequences — easy to reason about in isolation, surprising in aggregate.
