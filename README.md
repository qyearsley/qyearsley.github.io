# qyearsley.github.io

Personal website showcasing educational games, JavaScript experiments, and Chinese language tools.

## Contents

- **[Games](games/)** - Educational web games (math, reading, cybersecurity)
- **[JavaScript Experiments](javascript/)** - Interactive tools and visualizations
- **[Chinese Tools](chinese/)** - Language learning resources

## Development Commands

```bash
npm install     # Install dependencies
npm start       # Start local development server (http://localhost:8000)
npm test        # Run tests
npm run lint    # Run all linters (JS, HTML, CSS)
npm run format  # Format code with Prettier
```
